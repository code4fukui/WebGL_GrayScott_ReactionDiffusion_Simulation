# WebGLによるグレイ・スコット反応拡散シミュレーション

本プロジェクトは、様々なパターンの形成をシミュレートするグレイ・スコット反応拡散モデルをWebGLで実装したものです。

## デモ
[https://code4fukui.github.io/WebGL_GrayScott_ReactionDiffusion_Simulation/](https://code4fukui.github.io/WebGL_GrayScott_ReactionDiffusion_Simulation/)

## 特徴
- WebGLアクセラレーションによるグレイ・スコット反応拡散モデルのシミュレーション
- 拡散率、補充率 (feed)、消滅率 (kill)、時間ステップ、空間ステップのパラメータ調整機能
- 様々な反応タイプのプリセットパターン
- 2Dおよび3Dの可視化モード
- FPS表示によるリアルタイムのパフォーマンスモニタリング

## 使用方法
1. リポジトリをクローンします:
```
git clone https://github.com/code4fukui/WebGL_GrayScott_ReactionDiffusion_Simulation.git
```
2. Webブラウザで `index.html` ファイルを開きます。
