# WebGL Gray-Scott 反応拡散シミュレーション

WebGLを使った Gray-Scott 反応拡散モデルのリアルタイムシミュレーションです。さまざまな反応パターンを生成することができます。

## デモ
[https://code4fukui.github.io/WebGL_GrayScott_ReactionDiffusion_Simulation/](https://code4fukui.github.io/WebGL_GrayScott_ReactionDiffusion_Simulation/)

## 機能
- Gray-Scott 反応拡散モデルのリアルタイムシミュレーション
- 拡散係数、給餌率、殺傷率などのパラメータ調整
- 事前設定された反応パターンの選択
- 2Dと3Dの描画モード切り替え
- FPS表示による実行速度の監視

## 使い方
ブラウザでデモサイトを開くと、シミュレーションが表示されます。
GUIを使ってパラメータを調整できます。
